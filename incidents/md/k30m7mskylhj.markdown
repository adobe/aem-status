## Postmortem

*Posted May 06, 2022 - 08:14 UTC*

Between 07:04 and 07:11 UTC, a small amount of errors originating from the OneDrive API bubbled up to end users through the Helix Admin API and caused errors to be displayed in the Helix Sidekick. Operations are back to normal, affected users can reload their browser tab.

No further action will be taken at this point apart from continuing to closely monitor the situation.

This incident has been resolved.
